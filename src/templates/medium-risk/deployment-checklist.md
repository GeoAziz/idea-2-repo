# Deployment Checklist

## Pre-Deployment
- [ ] Verify tests pass
- [ ] Confirm environment variables are set
- [ ] Review dependency updates
- [ ] Validate backup/rollback plan

## Deployment
- [ ] Deploy to staging
- [ ] Run smoke tests
- [ ] Monitor logs and metrics

## Post-Deployment
- [ ] Confirm core flows
- [ ] Monitor error rates
- [ ] Announce release
